# vicibox-recording-extraction-bulk
<hr>
<b>Bulk recording file extraction from backend of a vicidial server from 8.0.1 to 11.0.1 or later.</b>
<hr>

Usage :- 

<br>Step 1 --> Clone the repository on the server.
```
git clone https://github.com/samsepoil1211/vicibox-recording-extraction-bulk.git
```
<br>
<br>Step 2 --> Navigate to the cloned directory.

```
cd vicibox-recording-extraction-bulk
```
<br>
<br>Step 3 --> Inside the directory, create a .txt file which will contain the bulk numbers without 1 and the arrangement will be one after other.<br>
<br>Step 4 --> Give proper permission for the script with this command 

```
chmod +x rec_extractor.sh
```
<br>
<br> Step 5 --> Once the script runs, you will get a directory named "extracted recordings" in the root directory. Inside that directory, the recordings are stored as per sub-folder wise for better understanding.<br>
